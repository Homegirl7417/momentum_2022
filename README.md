# momentum_2022 

### 간단 설명
- Vanilla JS로 만든 Momentum 크롬 확장앱 클론 프로젝트입니다.
- 모바일, PC 반응형으로 개발중인 프로젝트입니다.

### 제공 기능
- 사용자의 이름을 저장합니다.
- 위치 정보를 통해 날씨와 기온을 알려줍니다.
- 오늘 해야되는 가장 중요한 일을 기록합니다.
- 작업 리스트를 만들고 관리할 수 있습니다.
- 랜덤한 배경화면과 명언을 제공합니다.
